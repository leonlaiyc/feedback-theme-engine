# Statistical Signal Layer

- Separate prevalence from rating association and business priority.
- Include effect sizes where possible, not only p-values.
- Use confidence intervals or uncertainty estimates for theme metrics.
- Apply multiple testing correction when comparing many themes.
- Distinguish exploratory findings from confirmatory claims.
- Preserve enough evidence to trace each theme statistic back to source reviews.
- Avoid overstating causal or company-specific conclusions.
