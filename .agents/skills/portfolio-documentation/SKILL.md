# Portfolio Documentation

- Write for recruiters, hiring managers, and technical interviewers.
- Keep framing professional, concise, and honest.
- Explain business value without inflated claims.
- Make the methodology defendable in an interview.
- State limitations clearly.
- Avoid claiming production readiness or real company conclusions.
- Tie insight claims back to evidence and documented methodology.
